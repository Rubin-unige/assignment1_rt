\doxysubsection{Detailed Description}
This node monitors the distance between two turtles and checks if they are near boundaries.

Subscribes to\+:
\begin{DoxyItemize}
\item /turtle1/pose (turtlesim/\+Pose)\+: The pose of turtle1.
\item /turtle2/pose (turtlesim/\+Pose)\+: The pose of turtle2.
\end{DoxyItemize}

Publishes to\+:
\begin{DoxyItemize}
\item /turtle1/cmd\+\_\+vel (geometry\+\_\+msgs/\+Twist)\+: Velocity commands for turtle1.
\item /turtle2/cmd\+\_\+vel (geometry\+\_\+msgs/\+Twist)\+: Velocity commands for turtle2.
\item /turtle\+\_\+distance\+\_\+topic (assignment1\+\_\+rt/turtle\+\_\+distance)\+: Custom message containing distance and boundary information.
\end{DoxyItemize}

Description\+: This node calculates the distance between turtle1 and turtle2 and checks if they are too close or near the boundaries. If the turtles are too close or near the boundaries, the node stops them by publishing zero velocity commands. It also publishes a custom message containing the distance and boundary status of the turtles.
